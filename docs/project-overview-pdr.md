# Agent Bus — Product Development Requirements

## Vision

Bridge AI coding sessions to 3D visualization at zero inference cost. Any Claude Code session, anywhere, rendered as a living agent in Claw3D's retro office.

## Problem Statement

Claude Code sessions are invisible to external visualization tools. Claw3D only renders OpenClaw agents. No bridge exists to show Claude Code activity (tool use, file edits, task completion) as 3D agent behavior.

## Target Users

- Developers running multiple Claude Code sessions across machines
- Teams wanting visual dashboards of AI agent activity
- Anyone using Claw3D who wants to see non-OpenClaw agents

## Functional Requirements

| Req | Description | Priority |
|-----|-------------|----------|
| FR-1 | WebSocket hub accepts events from producers | P0 |
| FR-2 | POST /events HTTP endpoint for hook-based producers | P0 |
| FR-3 | Broadcast events to all connected WebSocket consumers | P0 |
| FR-4 | JSONL file logging for persistence and replay | P1 |
| FR-5 | Claw3D adapter translates events to Claw3D WebSocket protocol | P0 |
| FR-6 | Multi-project isolation via project field in events | P1 |
| FR-7 | Replay past events from JSONL log | P2 |
| FR-8 | CLI generated by CLI-Anything with Skill.md | P1 |

## Non-Functional Requirements

| Req | Target |
|-----|--------|
| NFR-1 | < 50ms event latency (publish → Claw3D render) |
| NFR-2 | < 30MB memory at steady state |
| NFR-3 | Zero LLM API token cost |
| NFR-4 | Works over LAN and Tailscale |

## Success Criteria

- [ ] Claude Code hook fires → agent appears in Claw3D 3D office
- [ ] Multiple projects show isolated agents
- [ ] JSONL log captures all events for replay
- [ ] CLI-Anything generates working CLI with Skill.md
- [ ] Works across Mac Mini ↔ VPS via Tailscale

## Constraints

- No Claw3D source modifications (use existing WebSocket protocol)
- OpenClaw stays passive ($0 tokens)
- Must work with Claude Max subscription (no API keys required)

## Out of Scope (v1)

- NATS/Redis/MQTT transport (future phase)
- nanobot/ClawWork integration
- AI-Trader signals
- Authentication/authorization between machines
